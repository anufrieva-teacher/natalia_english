# Сайт Наталии Сергеевны Ануфриевой

Статический сайт для GitHub Pages.

## Как обновить фото
Замените файлы с теми же именами:

- `assets/images/hero.jpg` — фото на главной
- `assets/images/about.jpg` — фото на странице «Обо мне»
- `assets/images/avatar.jpg` — квадратный аватар
- `assets/images/lesson-board.jpg` и `assets/images/language-board.jpg` — скриншоты уроков/Miro

Если имя файла не менять, интерактивные редактировать не нужно.

## Как добавить материал
Добавьте запись в `data/materials.json`.

## Как добавить тренажёр
1. Положите папку тренажёра в `trainers/название-тренажера/`.
2. Добавьте запись в `data/trainers.json`.
